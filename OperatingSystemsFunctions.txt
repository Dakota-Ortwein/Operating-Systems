The device manager is the portion of the operating system that interfaces with hardware devices. These devices can range from a monitor to an external hard drive. This also includes the computer's hard disks.
The memory manager is in charge of the silicone and RAM memory in a computer. This is not the data and files written on a disk. That is handled by the file manager.
The file manager is the part of the operating system that is in charge of the files and programs written on a computer's disk. These can range from a text document to a software application. The file manager is what organizes and handles the operation of the files.
